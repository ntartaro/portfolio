Where Over the World is the ISS?

An interactive tracker for the International Space Station that uses the Google Maps API to display the ISS location in realtime. Users can sign-up and login to create custom locations and see their current distance to the ISS at any time.

Technologies used:

  • HTML5
  • CSS3
  • JavaScript
  • React
  • Express
  • Node.js
  • MongoDB
