Hangman

The first coding project I ever attempted, made entirely in vanilla JavaScript. A fully functional version of the classic game Hangman able to be played either solo or with two people. Has a random word mode that draws from a library of over 300,000+ words!

Technologies used:

  • HTML5
  • CSS3
  • JavaScript

Where Over the World is the ISS?

An interactive tracker for the International Space Station that uses the Google Maps API to display the ISS location in realtime. Users can sign-up and login to create custom locations and see their current distance to the ISS at any time.

Technologies used:

  • HTML5
  • CSS3
  • JavaScript
  • React
  • Express
  • Node.js
  • MongoDB

MusicTown Live

Music Town Live is an app where you can find and browse local music events and buy tickets for them. Users can search by genre, location, date, etc. for any concert they are interested in seeing. Mystery tickets will be a future addition, stay tuned!

Technologies used:

  • HTML5
  • CSS3
  • JavaScript
  • React
  • Firebase
